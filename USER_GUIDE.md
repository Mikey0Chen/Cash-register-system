## 🔧 解决方法（选择其一）

### 方法1：强制刷新（推荐）⭐

**Windows/Linux**
```
按住 Ctrl + Shift + R
```

**Mac**
```
按住 Cmd + Shift + R
```

这会强制浏览器重新下载所有文件，不使用缓存。

---

### 方法2：清除浏览器缓存

#### Chrome / Edge
1. 按 `Ctrl + Shift + Delete`（Mac: `Cmd + Shift + Delete`）
2. 选择"缓存的图片和文件"
3. 时间范围选择"过去 1 小时"或"全部时间"
4. 点击"清除数据"
5. 刷新页面（F5）

#### Firefox
1. 按 `Ctrl + Shift + Delete`（Mac: `Cmd + Shift + Delete`）
2. 勾选"缓存"
3. 时间范围选择"全部"
4. 点击"立即清除"
5. 刷新页面（F5）

#### Safari
1. 菜单栏 → 开发 → 清空缓存
2. 或按 `Cmd + Option + E`
3. 刷新页面（F5）

## 🔍 如何确认缓存已清除？

打开浏览器开发者工具（F12），切换到"网络"（Network）标签：

1. 勾选"禁用缓存"（Disable cache）
2. 刷新页面（F5）
3. 查看请求列表中 `Financial-B-xfX923.js` 的状态：
   - ✅ 状态码 `200` = 从服务器重新下载
   - ❌ 状态码 `304` = 仍在使用缓存
